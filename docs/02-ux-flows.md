# UX Flows (v0.1)

## Quick Capture
1) Shortcut opens modal with Tier 0 fields
2) Project defaults to last used
3) Paste error (preserve whitespace)
4) Save → toast “Saved. Add details?”
5) If details: open full form
6) On save: secret-scan + duplicate hint

## Retrieval
1) Search box (full-text across title, exact error, fix summary, root cause, repro)
2) Filters: problem type, project, language, framework, service, environment, date, severity, time spent
3) Quotes → exact phrase
4) Results show title, fix summary, chips, date + quick-copy actions
